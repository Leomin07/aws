2. Create user IAM


```
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "AllowPublicRead",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": [
                "arn:aws:s3:::test777676/*",
                "arn:aws:s3:::test777676"
            ]
        }
    ]
}

```
